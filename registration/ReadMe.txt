Things that need to be installed:
- Firbase Core
- Firebase Auth
these are installed through cocopods in the Podfile
- this application also has APN eneabled and has a developers certificate

How To use:
1. Press Login or Register to Login or Register.
2. Enter your phone number and request code
3. A code will be sent to the phone number through sms. Enter this code to move forward to the home screen.
